# ZKClaims
A platform for developers to get hired by claiming their skills using Reclaim protocol. <br />
It helps Companies to save their time by not going through so many application

## stacks used
<l1>Reclaim Protocol</li>
<li>NodeJS</li>
<li>ReactJS</li>
<li>MongoDB</li>
<li>ExpressJS</li>
